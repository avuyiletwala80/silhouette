  Size: ~5KB

WHAT EACH FILE DOES
================================================================================

FRONTEND FLOW:
  1. User opens index.html
  2. Browser loads styles.css (styling)
  3. Browser loads scripts.js (functionality)
  4. User interacts with website
  5. Forms send data to backend (if deployed)

BACKEND FLOW:
  1. Frontend sends form data to backend-server.js
  2. Backend validates and stores in Supabase (supabase-setup.sql)
  3. Backend sends emails via SendGrid
  4. Backend returns response to frontend
